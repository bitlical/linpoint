# Security Policy

## Supported Versions

Linpoint is pre-1.0. Security fixes are applied to the latest released version.

## Reporting a Vulnerability

Do not open a public issue for a suspected vulnerability. Use
[GitHub private vulnerability reporting](https://github.com/bitlical/linpoint/security/advisories/new)
and include the affected version, impact, reproduction steps, and any proposed
mitigation.

You should receive an initial response within seven days. Confirmed issues will
be coordinated privately until a fix and disclosure are ready.
